TUTORIAL CSS3 - Fuentes y Efectos de texto

Propiedades para modificar las fuentes y efectos de los textos

Text-Decoration

Propiedades que se utiliza para controlar la decoración del texto en un elemento. Se puede aplicar diferentes tipos de decoración, como subrayado, tachado, línea sobre el texto, etc.

Sintaxis

text-decoration-line: none|overline|line-through|underline;
text-decoration-color: color;
text-decoration-style: dashed|dotted|double|solid|wavy;
text-decoration-thickness: grosor;

Donde pueden haber estos valores:

  • line (obligatoria): Posición de la línea.
    Algunos valores son:
    • none: Elimina la línea.
    • overline: Línea superior.
    • line-through: Línea en medio (tachado).
    • underline: Línea inferior (subrayado).
  • color (opcional): Color de la línea. Puede ser su nombre, su valor hexadecimal o su rgb(a).
  • style (opcional): Estilo de la línea.
    Algunos valores son:
    • dashed: Línea de líneas.
    • dotted: Línea de puntos.
    • double: Línea doble.
    • solid: Línea sólida.
    • wavy: Línea ondulada.
  • thickness (opcional): Grosor de la línea. Valor númerico (px, rem, etc.) o porcentaje.
Esto se puede juntar de la siguiente forma: 
text-decoration: line-through brown wavy 25%;

Ejemplos:

text-decoration-line: overline;

Texto con línea arriba

text-decoration-line: underline;
text-decoration-color: brown;

Texto con subrayado marrón

text-decoration-line: line-through;
text-decoration-style: wavy;

Texto tachado con línea ondulada

text-decoration-line: underline;
text-decoration-thickness: 25%;

Texto con subrayado grueso

text-decoration: line-through brown wavy 25%;

Texto con una línea con varias propiedades

En esta clase usamos text-decoration con todas las propiedades. Este código equivaldría al siguiente: 
text-decoration-line: line-through;
text-decoration-color: brown;
text-decoration-style: wavy;
text-decoration-thickness: 25%;

Text-Transform

Indica que si el texto esta en mayuscula o en minuscula o en capital

Sintaxis

text-transform: uppercase|lowercase|capitalize;

Admite tres opciones:

  • uppercase: Todo el texto esta en mayuscula.
  • lowercase: Todo el texto esta en minuscula.
  • capitalize: La primera letra de cada palabra del texto esta en mayuscula

Ejemplos:

text-transform: uppercase;

texto en mayuscula

text-transform: lowercase;

texto en minuscula

text-transform: capitalize;

texto en capital

Text-Shadow

Indica sombreado que va a tener el texto

Sintaxis

text-shadow: 3px 3px red;

Admite de dos a cuatro valores, que son los siguientes:

  • Dos valores: Se aplica el primero al sombreado horizontal y el segundo al sombreado vertical
  • Tres valores: El primer valor se aplica al sombreado horizontal, el segundo al sombreado vertical y el tercer valor al color que tiene el shadow
  • Cuatro valores: El primer valor se aplica al sombreado horizontal, el segundo al sombreado vertical, el tercero añade un efecto y el ultimo valor indica el color

Ejemplos:

text-shadow: 3px 3px;

Text-shadow1

text-shadow: 3px 3px red;

Text-shadow2

text-shadow: 3px 3px 4px red;

Text-shadow3

Text-Overflow

Propiedad que indica el contenido desbordado que no se muestra.

Sintaxis

text-overflow: clip|ellipsis;

Admite un valor, que puede ser uno de los siguientes:

  • clip: Este valor recortará el texto.
  • ellipsis: Este valor sustituirá el texto por una elípsis (...).

Ejemplos:

Texto recortado:

text-overflow: clip;

Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestiae velit laudantium adipisci esse voluptates deserunt eum ex, excepturi, amet, sequi praesentium fugiat fugit accusamus vel nostrum numquam neque repellat autem.

Aquí tenemos una de las clases de los ejemplos anteriores. Usamos overflow: hidden para ocultar el el texto que sobresalga. Al final ponemos text-overflow para indicar que hacer con el overflow, con "clip" recortamos el texto y con "ellipsis" ponemos el texto con una elípsis que serían los tres puntitos.

Texto con elípsis:

text-overflow: ellipsis;

Lorem ipsum dolor sit amet consectetur adipisicing elit. Aperiam quisquam repellat earum recusandae ad voluptates necessitatibus alias eaque unde expedita, iste, at quibusdam tenetur quis voluptatibus harum culpa a porro.

Y con el valor "break-word" se separará la palabra a la siguiente línea.

Ejemplos:

Texto separado:

Word-Break

Propiedad que establece las normas del salto de línea.

Sintaxis

word-break: keep-all|break-all;

Admite un valor, que puede ser uno de los siguientes:

  • keep-all: Este valor establecerá el salto de línea en los guiones o espacios.
  • break-all: Este valor establecerá el salto de línea en los caractéres.

Ejemplos:

Salto de línea con keep-all (por guiones/espacios):

word-break: keep-all;

This paragraph contains some text. This line will-break-at-hyphens.

Esta clase utliza word-break con el valor keep-all para establecer el salto de línea en los guiones y espacios.

Salto de línea con break-all (por letras):

word-break: break-all;

This paragraph contains some text. The lines will break at any character.

Writing-Mode

Propiedad que establece la orientación del texto.

Sintaxis

writing-mode: horizontal-tb|vertical-rl|vertical-lr|sideways-rl|sideways-lr;

Admite un valor, que puede ser uno de los siguientes:

  • horizontal-tb: Este valor establecerá la orientación de manera horizontal.
  • vertical-rl: Este valor establecerá la orientación de manera vertical (de derecha a izquierda).
  • vertical-lr: Este valor establecerá la orientación de manera vertical (de izquierda a derecha).
Otros valores permitidos son sideways-rl, sideways-lr que pueden servir para textos árabes o hebreos.

Ejemplos:

Texto Horizontal:

writing-mode: horizontal-tb;

This paragraph contains some text. This line will-break-at-hyphens.

Texto Vertical (de derecha a izquierda):

writing-mode: vertical-rl;

This paragraph contains some text. The lines will break at any character.

Texto Vertical (de izquierda a derecha):

writing-mode: vertical-lr;

This paragraph contains some text. The lines will break at any character.

En esta clase hemos puesto el modo de escritura de manera vertical y que sea de izquierda a derecha.

Valor tb:

writing-mode: tb;

This paragraph contains some text. The lines will break at any character.

Text-Rendering

Propiedad que controla cómo se procesa y renderiza el texto en un elemento. Tiene varios valores posibles que afectan la calidad y la legibilidad del texto.

Sintaxis

text-rendering: optimizeSpeed|optimizeLegibility|geometricPrecision;

Admite un valor, que puede ser uno de los siguientes:

  • optimizeSpeed: Se centra en la velocidad de renderización, sacrificando la calidad visual.
  • optimizeLegibility: Se centra en la legibilidad del texto, incluso a expensas de la velocidad.
  • geometricPrecision: Se centra en renderizar el texto con precisión geométrica (Útil con gráficos vectoriales).

Ejemplos:

Velocidad optimizada:

text-rendering: optimizeSpeed;

Voluptatibus harum culpa a porro.

Legibilidad optimizada:

text-rendering: optimizeLegibility;

Voluptatibus harum culpa a porro.

Precisión geométrica:

text-rendering: geometricPrecision;

Voluptatibus harum culpa a porro.

En esta clase hemos colocado la renderización a "Precisión geométrica" aunque no hay mucha diferencia debido a que es un texto normal y no uno vectorial.

Text-Orientation

Propiedad que se utiliza para establecer la orientación del texto en un elemento. Puede definir si el texto se mostrará horizontal o verticalmente, y en qué dirección. Solo funciona si el writing-mode está puesto como texto vertical.

Sintaxis

text-orientation: mixed|upright|sideways|sideways-right;

Admite un valor, que puede ser uno de los siguientes:

  • mixed: Este valor indica que la orientación del texto es mixta, lo que significa que puede contener tanto texto vertical como horizontal.
  • upright: Este valor se utiliza para especificar que el texto debe mostrarse en posición vertical.
  • sideways: Indica que el texto debe mostrarse en posición horizontal, pero con la letra en posición vertical.
  • sideways-right: Similar a sideways, pero la orientación de la letra es hacia la derecha(izquierda).

Ejemplos:

Texto en posición vertical:

text-orientation: upright;

Font-Style

Hay tres tipos de letras :

  • Normal: Letra normal sin ninguna modificación.
  • Italic: Letra italica
  • Oblique: Similar a italica

Ejemplos:

Letra Normal

font-style: oblique;

Letra oblique

Font-Weight

Indica la forma de la letra

Sintaxis

font-weight: Normal|Bold|Bolder|Lighter;

Admite estas opciones:

  • Numero: El valor se introduce numerico. Desde 100 a 600 la letra se veria normal. A partir de 700 se veria en negrita
  • Normal: La letra se ve normal
  • Bold: La letra se ve en negrita
  • Bolder: La letra se ve más negrita de lo normal
  • Lighter: La letra se ve muy fina

Ejemplos:

font-weight: bold;

negrita

font-weight:bolder;

bolder

font-weight:lighter;

lighter

Font-Size

Indica que el tamaño de la fuente

Sintaxis

font-size: 10em;

Admite tres opciones:

  • Px: Se aplica el valor en pixeles.
  • Em: Se aplica el valor en em(1em equivale a 16px)
  • %: Se aplica el valor en porcentaje(aumenta el tamaño al porcentaje indicado con respecto al contenedor al que se aplique)

Ejemplos:

font-size: 20px;

20 pixeles

font-size: 1em;

1 em

font-size: 50%;

50%

Font-Stretch

Es una propiedad del font-face para indicar como van a estar las letras de esa fuente

Sintaxis

font-stretch: condensed|expanded|ultra-expanded|%;

Admite 4 opciones:

  • Condensed: Pone las letras condensadas entre si
  • Expanded: Expande las letras
  • ultra-expanded: Mucho mas expandidas que en el expanded
  • %: Situa las letras con un espacio que se le haya indicado

Ejemplos:

font-stretch: condensed;

Este parrafo va a estar condesado
Esta linea esta pegada

font-stretch: expanded;

Este parrafo va a estar expandido
Esta linea esta separada

font-stretch: 50%

Este parrafo estara un 50% pegado
Esta linea estara ese 50%
