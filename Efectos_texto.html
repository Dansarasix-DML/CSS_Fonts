text-overflow

Propiedad que indica el contenido desbordado que no se muestra.

Sintaxis

p {text-overflow: valor;};

Admite un valor, que puede ser uno de los siguientes:

  • clip: Este valor recortará el texto.
  • ellipsis: Este valor sustituirá el texto por una elípsis (...).

Ejemplos:

Texto recortado:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestiae velit laudantium adipisci esse voluptates deserunt eum ex, excepturi, amet, sequi praesentium fugiat fugit accusamus vel nostrum numquam neque repellat autem.

Texto con elípsis:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Aperiam quisquam repellat earum recusandae ad voluptates necessitatibus alias eaque unde expedita, iste, at quibusdam tenetur quis voluptatibus harum culpa a porro.

Ejemplo de código:

.fonts_text_effects_test1 {
    white-space: nowrap;
    width: 200px;
    border: 1px solid #000000;
    overflow: hidden;
    text-overflow: clip;
}
Aquí tenemos una de las clases de los ejemplos anteriores. Usamos overflow: hidden para ocultar el el texto que sobresalga. Al final ponemos text-overflow para indicar que hacer con el overflow, con "clip" recortamos el texto y con "ellipsis" ponemos el texto con una elípsis que serían los tres puntitos.

word-wrap

Propiedad que divide palabras largas y las ajusta en la siguiente línea.

Sintaxis

p {word-wrap: break-word;}

Y con el valor "break-word" se separará la palabra a la siguiente línea.

Ejemplos:

Texto sin separar:

This paragraph contains a very long word: thisisaveryveryveryveryveryverylongword. The long word will break and wrap to the next line.

Texto separado:

This paragraph contains a very long word: thisisaveryveryveryveryveryverylongword. The long word will break and wrap to the next line.

Ejemplo de código:

.fonts_text_effects_test4 {
    width: 200px;
    height: 100px;
    border: 1px solid #000000;
    word-wrap: break-word;
}
Aquí en esta clase aumentamos la altura y haciendo uso de word-wrap y con el valor break-word pasamos la palabra larga partida a la siguiente línea.

word-break

Propiedad que establece las normas del salto de línea.

Sintaxis

p {word-break: valor;}

Admite un valor, que puede ser uno de los siguientes:

  • keep-all: Este valor establecerá el salto de línea en los guiones o espacios.
  • break-all: Este valor establecerá el salto de línea en los caractéres.

Ejemplos:

Salto de línea con keep-all (por guiones/espacios):

This paragraph contains some text. This line will-break-at-hyphens.

Salto de línea con break-all (por letras):

This paragraph contains some text. The lines will break at any character.

Ejemplo de código:

.fonts_text_effects_test5 {
    width: 150px;
    height: 100px;
    border: 1px solid #000000;
    word-break: keep-all;
}
Esta clase utliza word-break con el valor keep-all para establecer el salto de línea en los guiones y espacios.

writing-mode

Propiedad que establece la orientación del texto.

Sintaxis

p {writing-mode: valor;}

Admite un valor, que puede ser uno de los siguientes:

  • horizontal-tb: Este valor establecerá la orientación de manera horizontal.
  • vertical-rl: Este valor establecerá la orientación de manera vertical (de derecha a izquierda).
  • vertical-lr: Este valor establecerá la orientación de manera vertical (de izquierda a derecha).
Otros valores permitidos son lr, lr-tb, rl, rl-tb, tb, tb-rl que pueden servir para textos árabes o hebreos.

Ejemplos:

Texto Horizontal:

This paragraph contains some text. This line will-break-at-hyphens.

Texto Vertical (de derecha a izquierda):

This paragraph contains some text. The lines will break at any character.

Texto Vertical (de izquierda a derecha):

This paragraph contains some text. The lines will break at any character.

Valor tb:

This paragraph contains some text. The lines will break at any character.

Ejemplo de código:

.fonts_text_effects_test9 {
    width: 150px;
    height: 100px;
    border: 1px solid #000000;
    writing-mode: vertical-lr;
}
En esta clase hemos puesto el modo de escritura de manera vertical y que sea de izquierda a derecha.
